Descripción del Sistema
El sistema de gestión para el MiniMarket es una aplicación web que permite a los clientes realizar compras en línea y al dueño del negocio gestionar el catálogo, inventario y ventas de manera eficiente. La plataforma ofrece las siguientes funcionalidades:

Cliente:
Ver catálogo de productos disponibles
Buscar productos por nombre o categoría
Agregar productos al carrito de compras
Realizar pedidos en línea

Administrador:
Gestionar catálogo de productos (agregar, editar, eliminar)
Gestionar inventario (actualizar stock, precios, etc.)
Realizar ventas en línea y presencial
Gestionar pedidos y estados de entrega

La aplicación web está diseñada para ser intuitiva y fácil de usar, permitiendo a los clientes encontrar lo que buscan y realizar sus compras de manera rápida y segura. El dueño del negocio puede gestionar el sistema de manera eficiente, lo que le permite tomar decisiones informadas para mejorar el negocio.
